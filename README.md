## Calculo médio de aluno 
``
var nota1 = 10;
var nota2  = 8;
var nota3 = 9;
var nota4 = 7;
var media = (nota1 + nota2 + nota3 + nota4) / 4;
if ( media >= 8 ) { 
    console.log ("Aluno aprovado")

} else {
    console.log("Aluno em recuperação")

}
``
